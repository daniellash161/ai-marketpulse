נתוני שוק אמיתיים

נרות יומיים של BTC/USDT מ־Binance משנת 2020, מדד פחד וחמדנות, נפח ותזרים קונים. הכל לקריאה בלבד.

מודלי למידת מכונה

Prophet, ARIMA, XGBoost ו־RNN/LSTM לצד מודל אנסמבל משוקלל, עם השוואה כנה מול baseline.

🎯

השוואה ל־Polymarket

הסתברות מרומזת חיה משוק ניבויים אמיתי, ספר פקודות, והפער מול תחזית המודל.
